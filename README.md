# JobCopilot

JobCopilot is an agentic AI assistant that helps with job applications.

It is designed to:
- analyze job offers
- retrieve relevant profile evidence from long-term memory
- generate application and follow-up email drafts
- save application records
- propose reminders for follow-up actions

## Tech stack

- Streamlit
- LangChain
- LangGraph
- Anthropic
- FAISS
- Hugging Face embeddings
- Gmail API
- Google Calendar API
- Tavily Search
